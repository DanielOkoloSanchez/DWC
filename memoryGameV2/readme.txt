 MEMORY GAME 
*********************

Descripción:
MEMORYGAME es un juego de memoria en el que tendrás que buscar las parejas que son iguales en un tablero de las dimensiones que el usuario vea conveniente.

MANERA DE UTILIZACIÓN:

Para qué aplicación funcione correctamente necesitamos que los valores que introduzcas sean los adecuados es decir que correspondan con los siguientes requisitos 

- No puede introducir ni letras ni caracteres especiales 
- No pueden ser introducidos valores negativos ni 0 todos deben de ser más grandes que 0.
- Las casillas tienen que ser pares 
- No puedes usar menos de cuatro casillas
